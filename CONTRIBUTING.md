# Contributing to awesome-agents

Contributions are welcome! This list is for agents you can actually run — not just theory.

## What belongs here

- A working repo with a `README.md` that has a quickstart
- Something runnable in < 10 minutes from a fresh clone
- Clear description of what problem it solves

## What does not belong here

- Abandonware (last commit > 2 years ago)
- Pure demos with no reusable code
- Repos without any documentation

## How to contribute

1. Fork this repo
2. Add your entry to the right section — keep the same table format:
   ```
   | [repo-name](link) | One sentence: what it does | Stack | `quickstart command` |
   ```
3. Keep entries alphabetical within each section
4. Open a PR with the title: `add: repo-name — one line description`

## Sections

| Section | What goes here |
|---------|---------------|
| MCP Agents | MCP servers that expose tools to AI |
| A2A Agents | Agents using the Agent2Agent protocol |
| SAP + Enterprise | Agents integrating with enterprise systems |
| RAG + Knowledge | Retrieval-augmented agents |
| Multi-Agent Systems | Frameworks for coordinating multiple agents |
| LLM Tools + Infrastructure | Supporting libraries for production agents |

## New section?

If your entry does not fit any existing section, propose a new one in the PR description.
